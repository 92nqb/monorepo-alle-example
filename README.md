### Development view of a module

Operating systems:

```bash
# module-name shoulds be the name of a folder in ./packages/node_modules/<module-name>
$ MODULE_NAME=<module-name> npm run start
```

Windows:

```bash
# module-name shoulds be the name of a folder in ./packages/node_modules/<module-name>
$ set MODULE_NAME=<module-name>
$ npm run start
```
